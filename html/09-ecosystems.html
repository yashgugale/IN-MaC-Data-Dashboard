IN-MaC Education and Workforce Development Data Dashboard
This data dashboard is designed to visualize education and workforce data for the state of Indiana.
Various datasets can be visited using the navigation tabs on the left panel. The main purpose is to find trends, patterns and relationships amoung various data fields for better decision making and collaboration.
